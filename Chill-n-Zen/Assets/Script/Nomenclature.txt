Nomenclature de nommage:
=== Variable ===
Variable global privé => _nomVariablePrivee
Variable global public => nomVariablePublic
Variable local (fonction/method) => nomVariableLocal
Variable Paramètre (fonction/method) => nomParametre
Variable accesseur => NomVariableAccesseur

=== Autres ===
Method/Fonction => NomFonction
Event => OnNomEvenement

Nommage de Fonction/Script = Seul utilité
